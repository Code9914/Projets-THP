Check-list du Débutant en C# / C++

Coche chaque étape une fois maîtrisée pour suivre ta progression !

  • Comprendre les variables et les types de données
  • Maîtriser les opérateurs de base (+, -, *, /)
  • Utiliser les conditions (if, else)
  • Créer des boucles (for, while)
  • Comprendre les fonctions et leur utilisation
  • Apprendre les tableaux et listes
  • Gérer les erreurs de base
  • Notions de programmation orientée objet
